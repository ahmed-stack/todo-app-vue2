# Live Demo
[Todo App]("https://euphonious-pudding-7b5e6c.netlify.app/")

```bash
# Install Dependencies
$ npm install

# Run Application
$ npm run dev or npm start

# Build Application
$ npm run build

# Run the app
$ npm start

# Preview Application
$ go to http://localhost:5173/ to see live preview
```
